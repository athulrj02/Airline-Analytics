#  Airline Analytics Dashboard

An interactive data visualization project analyzing **flight delays and cancellations across the United States** using **Power BI and Tableau**.

This project was developed as part of a Data Visualization assignment, focusing on transforming raw airline performance data into meaningful business insights through dashboards.

---

##  Project Overview

The goal of this project is to explore patterns in airline delays and cancellations across different:

* Airlines
* Airports
* Regions
* Time (monthly trends)

The dashboard provides a user-friendly interface for exploring key delay metrics and identifying operational inefficiencies in the aviation sector.

---

##  Tools & Technologies

* **Power BI** – Interactive dashboard creation
* **Tableau** – Dashboard replication and comparison
* **Microsoft Excel** – Data cleaning and preprocessing
* **GitHub** – Version control and project hosting

---

##  Dataset

The dataset contains detailed flight records including:

* Airline and airport information
* Monthly flight statistics
* Delay categories:

  * Carrier delay
  * Weather delay
  * NAS (National Airspace System) delay
  * Security delay
  * Late aircraft delay
* Arrival delays and cancellations

The dataset includes **20+ columns**, meeting assignment requirements for complexity.

---

##  Key Visualizations

### 1. Airline Delay Percentage

Shows the percentage of delayed flights per airline, helping identify underperforming carriers.

### 2. Delay Across Regions

A geographic map visualizing delay distribution across U.S. airports.

### 3. Top 5 Airports by Cancelled Arrivals

Highlights airports with the highest number of cancellations.

### 4. Monthly Delay Trends

Tracks how delays (NAS & Weather) fluctuate throughout the year.

### 5. Combined Delay & Cancellation Trends

A dual-axis chart comparing:

* Arrival delays
* Flight cancellations

---

##  Dashboard Features

* Interactive **filters**:

  * Year selection
  * Airport selection
  * Delay type selection

* Dynamic exploration of:

  * Regional delay patterns
  * Airline performance
  * Seasonal trends

* Clean and structured layout for decision-making

---

##  Key Insights

* Certain airlines consistently show higher delay percentages
* Weather and NAS delays significantly impact peak months
* Major hub airports tend to experience higher cancellation volumes
* Delay patterns vary seasonally, with noticeable spikes during specific months

---

## 🔄 Tableau vs Power BI

This project was implemented in both tools to compare usability and capabilities:

| Feature       | Power BI                      | Tableau                         |
| ------------- | ----------------------------- | ------------------------------- |
| Ease of use   | Beginner-friendly             | More flexible but steeper curve |
| Visualization | Strong default visuals        | Highly customizable             |
| Performance   | Fast with structured datasets | Better for complex visuals      |
